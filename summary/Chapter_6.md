# Chapter 6. 간단한 웹 앱 만들기
> **아이덴티콘(identicon)**: 일반적으로 IP 주소나 사용자 이름의 해시 값을 이용하여 자동적으로 생성한 이미지를 아이덴티콘이라고 한다. 웹사이트의 경우 사용자 이름이나 IP 주소를 해싱하여 사용자 고유의 이미지를 제공하거나, 파비콘을 제공.

## 존재하는 이미지 이용하기
> **dnmonster**: RESTful API를 제공하는 도커 이미지

### 캐싱 추가하기
> **Redis**: 인-메모리 키-밸류 데이터 저장소. 크지않은 양의 정보를 처리하고, 안정성에 걱정할 필요가 없는 경우에 적합하다. 

## 마이크로서비스(microservice) 아키텍처
: 여러개의 작고 독립적인 서비스들로 시스템을 구성하는 것 
  <—> **모노리틱(monolithic)** 아키텍츠
